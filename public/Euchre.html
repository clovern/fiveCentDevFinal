The GUI for this project was made with Tkinter (a Python add-on), so it cannot be played directly on-site. However, you can play it yourself or check out the code at this github repository.

Instructions to play:

1: Download and unzip Euchre_AI repository from github link

2: If you do not have Python downloaded on your computer, download this from www.python.org

3: install the Python library Pillow. This can be done from the command line using pip install Pillow (pip) or conda install pillow (Anaconda/Miniconda).

4: Find the Euchre_final.py file in the Euchre_AI directory using your command line and run the file using python Euchre_final.py.
